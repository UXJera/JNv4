Website Iterations

Jeremiah Nyman version 1

Developed early 2013 This was the first attempt at building a website.
I had used previous CSS design and tailored the content to it.
I didn't understand how CSS worked at the time, and had no idea what responsive design was.
Near the end of this version of the website, I started working for a larger, more professional website and learned more of the intricacies of proper design work.

Jeremiah Nyman version 2

Developed July 2014 I had decided that I wanted to learn more about web design and take the things I've learned and create something meaningful. I had an idea of how I wanted to build the website, but still had many issues with trying to get things to work properly. I revamped the focus of the website towards a working portfolio, as opposed to what was more of a personal website that I made for myself and friends.

Jeremiah Nyman version 3

Developed Nov. 2014 This is the previous version of the current website, hand-coded through raw HTML and Sass without any frameworks (aside from a simple grid framework).
I had spent most of my time debugging and altering tiny bits of code to make things more responsive, but then I decided that I wanted to create a more interactive website built with a framework.
